# 🎬 Smart Movie Ticket Management System
A desktop-based movie ticket booking system developed using **Python, Tkinter, and MySQL**, integrated with a **Machine Learning recommendation system** to enhance user experience.

---

## 🚀 Features

* 🔐 User Registration & Login
* 🎥 View Movies & Show Timings
* 🎟️ Ticket Booking System
* 💺 Real-Time Seat Selection
* ❌ Booking Cancellation
* 🤖 Movie Recommendation System (ML-based)
* 🗄️ MySQL Database Integration

---

## 🛠️ Tech Stack

* **Frontend (GUI):** Tkinter / CustomTkinter / ttk
* **Backend:** Python
* **Database:** MySQL
* **Machine Learning:** Scikit-learn

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/movie-ticket-system.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Setup MySQL database

4. Run the project:

```
python main.py
```
